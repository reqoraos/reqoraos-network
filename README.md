# ReqoraOS

> **Run AI on Machines No One Owns. Decentralized GPU Compute for Everyone.**

ReqoraOS is a decentralized AI inference network built on Solana DePIN that aggregates idle GPUs from contributors around the world into a unified compute layer. Instead of relying on centralized cloud providers, ReqoraOS enables developers to access OpenAI-compatible inference endpoints while GPU providers earn rewards for verified compute contributions.

Designed for scalability, transparency, and permissionless participation, ReqoraOS transforms consumer GPUs into a distributed AI supercomputer where every verified inference strengthens the network. The project's public description emphasizes decentralized GPU pooling on Solana DePIN, OpenAI-compatible inference, and rewards for verified contributors. :contentReference[oaicite:0]{index=0}
